![home](./preview/home.png)
![view](./preview/view.png)


## Usage

在 `public/pdfs` 目录中创建下级目录作为分类，在分类目录中存放 PDF 文档。

运行 `npm run scan` 进行扫描，生成 PDF 索引文件。

运行 `npm run build` 进行构建。

## 辅助工具
